# KAMI

KAMI is a lightweight mobile-first 3D action duel. Drag anywhere, release to pierce, and finish a two-minute match before the lacquer dries.

## Play modes

- Solo duel against AI
- Online 1v1 rooms with invite links, powered by PeerJS/WebRTC
- Six-player bot rumble and 3v3 bot clash as expansion-ready rule sets

## Local development

```bash
npm install
npm run dev
```

## Build and test

```bash
npm run check
```
